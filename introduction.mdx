**roadbeat Studio Pro** extends the open-source [Studio Community Edition](https://docs.studio.roadbeat.dev) with 20 professional plugins for editorial workflows, advanced security, real-time collaboration, and enterprise integrations.

<Callout kind="info">
  Studio Pro plugins are loaded at runtime into the CE application. They never modify CE code — CE remains fully functional even if the Pro license expires.
</Callout>

## How It Works

Studio CE ships a **plugin-sdk** that defines extension points — controllers, event handlers, editor panels, and admin routes. Each Pro plugin implements this SDK and is loaded at runtime into the CE application. A signed **license key** (Ed25519 JWT) controls which plugins are activated.

<Steps>
  <Step title="Set license key" icon="key">
    Set the `ROADBEAT_LICENSE_KEY` environment variable with your Ed25519-signed JWT token.
  </Step>
  <Step title="Plugins load automatically" icon="plug">
    CE's `PluginLoaderService` scans the plugins directory and loads all licensed plugins at startup.
  </Step>
  <Step title="Pro features appear" icon="layout">
    New sidebar menu items, admin pages, content editor panels, and API endpoints become available.
  </Step>
</Steps>
